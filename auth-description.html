Авторизация

В моём проекте реализована простая и безопасная система авторизации пользователей через электронную почту и пароль.

Возможности:

  • Регистрация нового пользователя с проверкой email;
  • Вход с уже зарегистрированными данными;
  • Выход из аккаунта;
  • Сохранение сессии при повторном запуске приложения;
  • Обработка ошибок (неверный логин, отсутствие сети и т.п.).

Авторизация построена с использованием Firebase Authentication и соблюдает все актуальные требования к безопасности и удобству для пользователя.
